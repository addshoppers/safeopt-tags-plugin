Easily add SafeOpt tags on your site.

== Description ==
SafeOpt Tags manager plugin to easily and quickly add tags to your site.  This plugin works for WooCommerce platform.  

= BENEFITS =
* Never have to worry about inadvertently breaking your site by adding code
* Avoid inadvertently placing snippets in the wrong place
* Eliminate the need for other plugins to help manage Tags / Scripts
* Never lose SafeOpt Tags when switching or changing themes

= FEATURES =
* Work for SafeOpt Tags only
* Adds SafeOpt Tags to your site for the SafeOpt widget and conversion tracking

= INJECTION LOCATIONS =
1. Footer On all Pages
2. WooCommerce Thankyou Page

> #### Plugin Information
> * [SafeOpt](https://www.safeopt.com/)
> * [Plugin Documentation](https://www.safeopt.com/)


== Installation ==

1. Upload `SafeOpt Tags` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add Site ID provided to you by SafeOpt and click save changes
1. Visit your site and check for `AddShoppersWidget` in console

== Frequently Asked Questions ==

= Q. Where do I get my Site ID? =
A. Please contact Addshoppers at [help@addshoppers.com](mailto:help@addshoppers.com)

= Q. Where do I enter the Site ID? =
A. After activating the plugin, you can click on settings link under the plugin name OR you can click the SafeOpt Tags tab on the left side navigation. From there, you can enter the site id.
